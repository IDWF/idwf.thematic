idwf.thematic Installation
--------------------------

To install idwf.thematic using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``idwf.thematic`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        idwf.thematic

* Re-run buildout, e.g. with:

    $ ./bin/buildout
